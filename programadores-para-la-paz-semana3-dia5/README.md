# Programadores para la Paz – Semana 3 Día 5

## Actividad
Cliente, servidor, HTTP, GET, POST y primer servidor con Node.js.

## Objetivo
Comprender de forma inicial cómo se comunican un cliente y un servidor mediante HTTP, y ejecutar un servidor básico con Node.js.

## Enfoque ciudadano
Esta práctica se conecta con plataformas participativas. Una comunidad puede consultar información y también enviar propuestas. Una plataforma debe responder de forma clara, responsable y comprensible.

## Importante

Esta clase es introductoria.

No se trabajará Express.
No se trabajarán bases de datos.
No se trabajará autenticación.
No se trabajará despliegue.
No se trabajará seguridad avanzada.
No se trabajarán APIs complejas.

## Comando nuevo del día

npm start

Este comando ejecuta el servidor definido en package.json.

## Rama de trabajo

No trabajes directamente sobre main.

Cada estudiante debe crear una rama propia con el formato:

nombres_apellidos

Ejemplo:

maria_gomez
carlos_rodriguez
ana_martinez_perez

## Evidencia

La evidencia se registra únicamente con el push de tu rama personal en GitHub.

No debes entregar archivos adicionales.
No debes subir capturas.
No debes publicar enlaces en Moodle.
No debes enviar evidencias por otro medio.
