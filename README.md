# `programadores-para-la-paz` `🕊️`

> Repositorio educativo de aprendizaje progresivo en programación, organizado por semanas.
> Cada módulo combina teoría, práctica y reflexión crítica.

---

## `📁` Estructura del repositorio

```
programadores-para-la-paz/
├── semana1/   # Algoritmos básicos y pensamiento computacional
├── semana2/   # Terminal y comandos del sistema
├── semana3/   # Control de versiones con Git
├── semana4/   # Introducción a servidores con Node.js
├── semana5/   # Conceptos de APIs REST
├── semana6/   # Manejo de incidencias y pruebas de API
├── semana7/   # Generación y prueba de reportes
├── semana8/   # Buenas prácticas y seguridad en plataformas
└── semana9/   # Componentes y despliegue de servidores
```

Cada carpeta contiene archivos `.txt` de conceptos y reflexiones. Las semanas con desarrollo backend incluyen un `server.js` con su `package.json`.

---

| `#` | Tema principal | Archivos clave |

---

## `🚀` Cómo usar este repositorio

```bash
# Clonar el repositorio
git clone https://github.com/<usuario>/programadores-para-la-paz.git
cd programadores-para-la-paz

# Navegar a una semana con servidor
cd semana4
npm install
node server.js
```

> Cada semana es independiente. Puedes explorar los archivos `.txt` directamente
> o ejecutar el servidor de la semana correspondiente si deseas probar el código.

---

## `🌱` Propósito

Este repositorio forma parte del programa **Programadores para la Paz**, una iniciativa de formación tecnológica con enfoque social. El objetivo es desarrollar habilidades técnicas sólidas combinadas con pensamiento crítico y compromiso con el entorno.
