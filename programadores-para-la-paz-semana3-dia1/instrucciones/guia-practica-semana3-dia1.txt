Guía práctica – Semana 3 Día 1

Actividad:
Introducción a JavaScript: variables, decisiones y algoritmos simples.

Repositorio:
Repositorio dispuesto por el docente.

Importante:
No debes trabajar directamente en main.

Cada estudiante debe crear una rama personal con el formato:

nombres_apellidos

Ejemplo:
maria_gomez
juan_perez
laura_torres_garcia

Evidencia:
La evidencia de esta actividad es únicamente el push de tu rama personal a GitHub.

No debes entregar nada en Moodle.
No debes enviar capturas.
No debes subir archivos adicionales.
No debes publicar enlaces por otro medio.

Comando nuevo del día:
node archivo.js

Pasos obligatorios:

1. Clonar el repositorio

git clone URL_DEL_REPOSITORIO

2. Entrar a la carpeta del proyecto

cd NOMBRE_DEL_REPOSITORIO

3. Crear la rama personal

git checkout -b nombres_apellidos

4. Verificar la rama actual

git branch

La rama marcada con * debe ser tu rama personal, no main.

5. Ejecutar el primer algoritmo

node algoritmo-verificacion.js

6. Leer el resultado en la terminal

7. Modificar una variable en algoritmo-verificacion.js

Por ejemplo:
let informacionVerificada = false

8. Ejecutar de nuevo

node algoritmo-verificacion.js

9. Ejecutar el segundo algoritmo

node decision-comunitaria.js

10. Modificar una variable en decision-comunitaria.js

Por ejemplo:
let tieneLenguajeClaro = false

11. Ejecutar de nuevo

node decision-comunitaria.js

12. Completar los archivos:

semana3/conceptos-javascript.txt
semana3/bitacora-comando-dia1.txt
semana3/reflexion-decisiones.txt

13. Revisar cambios

git status

14. Agregar cambios

git add .

15. Crear commit

git commit -m "Algoritmos simples con JavaScript semana 3 dia 1"

16. Publicar la rama

git push -u origin nombres_apellidos
