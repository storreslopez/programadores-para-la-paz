# Programadores para la Paz – Semana 3 Día 3

## Actividad
Ciclos con while: repetir procesos de revisión comunitaria.

## Objetivo
Comprender qué es un ciclo while en JavaScript y cómo permite repetir instrucciones mientras se cumpla una condición.

## Enfoque ciudadano
Esta práctica se conecta con el seguimiento comunitario. Una comunidad no revisa información una sola vez: verifica fuentes, da seguimiento a propuestas y repite procesos para tomar decisiones más responsables.

## Importante

Esta clase es introductoria.

No se trabajará todavía servidor.
No se trabajará HTTP.
No se trabajará localhost.
No se trabajará Express.
No se trabajarán APIs.
No se trabajará JSON avanzado.
No se trabajarán bases de datos.
