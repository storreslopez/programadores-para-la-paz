# `programadores-para-la-paz`

> `📚` Proyecto colaborativo para el aprendizaje de programación con propósito social.

---

## `✨` Descripción general

`programadores-para-la-paz` es un repositorio educativo que reúne ejercicios, reflexiones y guías de aprendizaje para estudiantes de programación. Cada semana cuenta con módulos teóricos y prácticos diseñados para fortalecer habilidades técnicas, pensamiento crítico y trabajo colaborativo.

Este repositorio está organizado por semanas y días, con archivos de actividades, reflexiones y recursos para acompañar el proceso de aprendizaje.

---

## `🧭` Estructura del repositorio

- `semana1/` - Introducción a conceptos básicos y preguntas reflexivas.
- `semana2/` - Terminal, comandos, Git y ejercicios prácticos.
- `semana3/` - Conceptos de Git y reflexión sobre el aprendizaje.
- `programadores-para-la-paz-semana2-dia5/` - Clase específica del día 5 con recursos, checklist y la práctica de accesibilidad.
