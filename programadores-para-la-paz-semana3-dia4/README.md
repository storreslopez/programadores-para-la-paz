# Programadores para la Paz – Semana 3 Día 4

## Actividad
Arrays, for y for...of: recorrer listas de información comunitaria.

## Objetivo
Comprender qué es un array en JavaScript y cómo for y for...of permiten recorrer listas de datos, como propuestas, mensajes o fuentes de información.

## Enfoque ciudadano
Esta práctica se conecta con la organización de información comunitaria. Una comunidad puede recibir muchas propuestas, mensajes o fuentes, y una plataforma debe poder revisarlas de forma ordenada.

## Importante

Esta clase es introductoria.

No se trabajará todavía servidor.
No se trabajará HTTP.
No se trabajará localhost.
No se trabajará Express.
No se trabajarán APIs.
No se trabajará JSON avanzado.
No se trabajarán bases de datos.
No se trabajarán métodos avanzados de arrays.

npm run revisar
