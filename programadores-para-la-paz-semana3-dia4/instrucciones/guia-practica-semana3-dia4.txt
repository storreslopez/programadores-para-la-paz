Guía práctica – Semana 3 Día 4

Actividad:
Arrays, for y for...of: recorrer listas de información comunitaria.

Repositorio:
Repositorio dispuesto por el docente.

Importante:
No debes trabajar directamente en main.

Cada estudiante debe crear una rama personal con el formato:

nombres_apellidos

Ejemplo:
maria_gomez
juan_perez
laura_torres_garcia

Evidencia:
La evidencia de esta actividad es únicamente el push de tu rama personal a GitHub.

No debes entregar nada en Moodle.
No debes enviar capturas.
No debes subir archivos adicionales.
No debes publicar enlaces por otro medio.

Comando nuevo del día:
npm run revisar

Pasos obligatorios:

1. Clonar el repositorio

git clone URL_DEL_REPOSITORIO

2. Entrar a la carpeta del proyecto

cd NOMBRE_DEL_REPOSITORIO

3. Crear la rama personal

git checkout -b nombres_apellidos

4. Verificar la rama actual

git branch

La rama marcada con * debe ser tu rama personal, no main.

5. Ejecutar el comando nuevo del día

npm run revisar

6. Leer el array mostrado en la terminal

7. Modificar array-basico.js agregando una nueva comunidad

8. Ejecutar de nuevo

npm run revisar

9. Ejecutar el ejemplo con for tradicional

node for-basico.js

10. Modificar for-basico.js agregando una nueva comunidad al array

11. Ejecutar de nuevo

node for-basico.js

12. Ejecutar el ejemplo con for...of

node for-of-basico.js

13. Modificar for-of-basico.js agregando una nueva comunidad al array

14. Ejecutar de nuevo

node for-of-basico.js

15. Ejecutar lista de propuestas

node lista-propuestas.js

16. Agregar una nueva propuesta a la lista

17. Ejecutar de nuevo

node lista-propuestas.js

18. Ejecutar revisión de mensajes

node revision-mensajes.js

19. Agregar o modificar un mensaje comunitario

20. Ejecutar de nuevo

node revision-mensajes.js

21. Ejecutar revisión de fuentes

node revision-fuentes.js

22. Agregar una nueva fuente

23. Ejecutar de nuevo

node revision-fuentes.js

24. Ejecutar resumen comunitario

node resumen-comunitario.js

25. Agregar una nueva actividad comunitaria

26. Ejecutar de nuevo

node resumen-comunitario.js

27. Completar los archivos:

semana3/conceptos-arrays-for.txt
semana3/bitacora-comando-dia4.txt
semana3/reflexion-listas-participacion.txt

28. Revisar cambios

git status

29. Agregar cambios

git add .

30. Crear commit

git commit -m "Arrays for y for of en JavaScript semana 3 dia 4"

31. Publicar la rama

git push -u origin nombres_apellidos
