# Programadores para la Paz – Semana 3 Día 1

## Actividad
Introducción a JavaScript: variables, decisiones y algoritmos simples.

## Objetivo
Comprender desde cero qué es JavaScript, cómo se ejecuta un archivo con Node.js y cómo se construyen decisiones básicas usando let, const, if y else.

## Enfoque ciudadano
Esta práctica se conecta con la toma de decisiones responsables en entornos digitales. Antes de compartir información, publicar una propuesta o responder a una comunidad, una plataforma debe seguir una lógica clara.

## Importante

Esta clase es introductoria.

No se trabajará todavía servidor.
No se trabajará HTTP.
No se trabajará localhost.
No se trabajará npm.
No se trabajará Express.
No se trabajarán APIs.
No se trabajará JSON avanzado.
No se trabajarán bases de datos.

## Comando nuevo del día

node archivo.js

Ejemplo:

node algoritmo-verificacion.js

## Rama de trabajo

No trabajes directamente sobre main.

Cada estudiante debe crear una rama propia con el formato:

nombres_apellidos

Ejemplo:

maria_gomez
carlos_rodriguez
ana_martinez_perez

## Evidencia

La evidencia se registra únicamente con el push de tu rama personal en GitHub.

No debes entregar archivos adicionales.
No debes subir capturas.
No debes publicar enlaces en Moodle.
No debes enviar evidencias por otro medio.
