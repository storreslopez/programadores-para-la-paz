Responde con tus propias palabras.

1. ¿Qué es JavaScript?

Respuesta: JavaScript es un lenguaje de programación que permite añadir comportamiento dinámico a las páginas web y ejecutar scripts en entornos como Node.js.

2. ¿Qué permite hacer Node.js en esta clase?

Respuesta: Node.js permite ejecutar archivos JavaScript en la terminal para probar algoritmos y automatizar tareas sin usar un navegador.

3. ¿Qué hace el comando node algoritmo-verificacion.js?

Respuesta: Ejecuta el archivo `algoritmo-verificacion.js` y muestra en la terminal los mensajes relacionados con la verificación de información.

4. ¿Qué es una variable?

Respuesta: Una variable es un identificador que almacena un valor que el programa puede leer o modificar.

5. ¿Para qué se usa const?

Respuesta: `const` declara una variable cuyo valor no debe reasignarse durante la ejecución.

6. ¿Para qué se usa let?

Respuesta: `let` declara una variable cuyo valor puede cambiar posteriormente.

7. ¿Qué significa if?

Respuesta: `if` ejecuta un bloque de código solo si se cumple una condición booleana.

8. ¿Qué significa else?

Respuesta: `else` define la alternativa que se ejecuta cuando la condición del `if` es falsa.

9. ¿Por qué una decisión en programación se parece a una decisión responsable en internet?

Respuesta: Porque ambas requieren evaluar evidencia y consecuencias antes de actuar; tomar decisiones informadas reduce riesgos y evita daño.

10. ¿Cómo se relaciona este ejercicio con la verificación de información?

Respuesta: El ejercicio muestra la importancia de comprobar hechos antes de compartir, usando condiciones que impiden acciones hasta confirmar la veracidad.
